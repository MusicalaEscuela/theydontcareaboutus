# Samples Musicala opcionales

Puedes copiar aquí los WAV del secuenciador original si quieres que el proyecto no dependa de la URL pública:

- bombo.wav
- redoblante.wav
- platillo.wav

Si no están, `audio.engine.js` intenta cargarlos desde:
https://musicalaescuela.github.io/secuenciadormusicala/assets/sounds/

Si tampoco cargan, usa sonidos sintéticos de respaldo.
